🚀 Project Title

FoodTrends – Customer Preferences Analysis

📌 Problem Statement

Understanding customer preferences in the food industry is challenging due to scattered data across surveys and social media.

This project aims to:

Analyze customer behavior
Identify trending food categories
Understand sentiment from social media

📊 Data Sources
Structured Data: Google Forms survey
Unstructured Data: Facebook comments scraped using Apify

🛠️ Tools & Technologies
Python (Pandas, NLP, TextBlob/VADER)
Power BI
Apify (Web Scraping)
Excel

🔄 Project Workflow
Data Collection
Collected survey data
Scraped Facebook comments using Apify
Data Cleaning
Removed duplicates
Handled missing values
Standardized categories
Exploratory Data Analysis (EDA)
Category-wise preferences
Customer demographics insights
Sentiment Analysis (NLP)
Classified comments into Positive / Negative / Neutral
Identified trending opinions
Dashboard Development
Built interactive Power BI dashboard

📈 Key Insights
🍕 Most preferred category: Fast Food
😊 Majority sentiment: Positive (65%+)
👥 Young customers prefer online ordering
⭐ High-rated products correlate with positive sentiment
